Warum dieser Umweg?

Tor verschleiert deine echte IP-Adresse, indem alle Anfragen über drei zufaellige Server weltweit geroutet werden. Niemand auf der Strecke sieht gleichzeitig Absender UND Ziel.

Bei der Onion-Adresse (.onion) bleibt der Traffic sogar komplett im Tor-Netz. Cloudflare und unser Server sehen NICHTS von dir, nicht mal welcher Tor-Exit du nutzt.

Ohne Tor: deine IP -> Cloudflare -> Hetzner. Beide könnten theoretisch loggen wer wann wo hier war. Mit Tor: technisch unmöglich.

Was wir schon machen: Backend speichert KEINE IP, KEINEN Zeitstempel auf Sekunden-Ebene, KEINEN Username. Nur kryptografische Adressen + verschlüsselte Pakete. Tor-Zugang ist das letzte Stueck damit es wirklich anonym ist.
